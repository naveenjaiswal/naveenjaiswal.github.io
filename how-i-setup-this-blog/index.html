How I setup this blog

01 April 2016

This blog is hosted freely on github published by ghost Blogging platform and buster.

install and setup ghost on ubuntu

  • create a folder for your blog

    sudo mkdir /workspace

  • download ghost link

    wget https://ghost.org/zip/ghost-0.7.8.zip

    change link according to your ghost current version

  • unzip and go to the directory

  • before installing ghost, first install nodejs

    sudo apt-get install nodejs

  • install ghost

    sudo npm install

  • start ghost locally

    sudo npm start

  • visit http://localhost:2368/ to create account, setup blog and publish posts.
  • edit config.js and change ur github url or custom url with default url in production settings.

setup github and buster pages

  • Go to your gitHub account and create a public repo named accountname.github.io (in my case it was naveenjaiswal.github.io)
  • switch to other folder whre you want to generate your static files

  • setup buster

    sudo buster setup

    and then provide git url https://github.com/accountname/accountname.github.io.git

  • generate the needed static files

    buster generate --domain=http://127.0.0.1:2368

  • switch to static folder and add,commit and push to git repo.

    cd static sudo git add --all sudo git commit -m "first commit" sudo git push

  • if you have custom domain , do necessary setting otherwise your domain live on accountname.github.io like this http://naveenjaiswal.github.io/
